Mac安装win7后无法引导

发表于 2016-04-20   |   分类于 运维   |  

Problem

Mac通过Boot Camp制作的安装U盘安装win7后重启无法引导，提示\windows\system32\drivers\AppleSSD.sys

Solution

这个原因是因为苹果最新版的Boot Camp提供的驱动是为了适配windows10，所以会造成安装win7会提示证书验证失败。

Solution for already installed

  1. 重启电脑进入pe
  2. 进入/windows/system32/drivers目录
  3. 删除AppleSSD.sys
  4. 重启电脑
  5. 下载Boot Camp v5.1.5640并安装

Solution for ready to install

  1. 制作安装盘
  2. 下载boot Camp v5.1.5640
  3. 删除安装盘$WinPEdrive$目录下AppleSSD文件夹
  4. 使用v5.1.5640/bootscamp/drivers下内容替换安装盘
  5. 重启进行安装
